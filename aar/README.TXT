DESCRIPTION:

This is the CAEN RFID library for android. You must include this aar
in your project. More information on www.caenrfid.com

SYSTEM REQUIREMENTS:
This library is compiled with Android Studio 2.2.2 and support GingerBread 3.1.0
or higher.
